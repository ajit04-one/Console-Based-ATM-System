# Console-Based-ATM-System
This is a console based application that runs in the terminal and interacts with the user through input/output files.

#Features
-Has PIN authentication with limited attempts
-Deposit and withdraw money
-Shows account balance using file handling
-Transaction logging
-Input validation for numeric values

#Files Used
-"account.txt"--> It stores current balance
- "transactions.txt" -->stores withdrawal history

#Concepts used in this program
-Class And Constructors
-File Handling (fstream)
-Input Validation
-Control flow and loops

#How to run

Compile using:
```bash
g++ atm.cpp -o atm
./atm
